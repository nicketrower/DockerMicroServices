Getting Started

Unzip the archive and find out the folder www. There are all the template files in this folder. You can open the following files for viewing in browser.

  • /css — folder with css files.
  • /fonts — folder with icon fonts.
  • /icon — folder with favicons.
  • /img — folder with image files.
  • /js — folder with javascript files.

  • index.html — home page.
  • artists.html — artists page.
  • artist.html — artist page.
  • news.html, news-masonry.html — different news pages.
  • article.html — article page.
  • events.html — events page.
  • event.html — event page.
  • releases.html — releases page.
  • release.html — release page.
  • store.html — store page.
  • item.html — item page.
  • cart.html — cart page.
  • contacts.html — contacts page.
  • signin.html — signin page.
  • signup.html — signup page.
  • forgot.html — password recovery page.
  • 404.html — 404 page.

Upload the template files (inside the folder www) to server with the help of one of the FTP-clients.

HTML structure

In my theme I use Bootstrap grid, if you don't know how to use it, here is a detailed information on its use.

The standard section has the following HTML structure:

<section class="section">
  <div class="row row--grid">

    <div class="col-xs-12">
      <div class="section__title clearfix">
        <h2>New Releases</h2>
        <a class="section__more" href="releases.html">See All</a>
      </div>
    </div>

    <div class="col-xs-6 col-sm-3">
      <article class="release-post">
        <figure class="release-post__cover"><a href="release.html"><img src="img/covers/cover1.jpg" alt=""></a></figure>

        <header class="release-post__header">
          <h2 class="release-post__title"><a href="release.html">Forbidden Forest</a></h2>
          <a href="artist.html" class="release-post__autor">Chris S.</a>
        </header>
      </article>
    </div>

  </div>
</section>

Grids

Default bootstrap grid have next structure:

<div class="container">
  <div class="row">
	...
  </div>
</div>


If you add to "row" modification "row--grid", for example:

<div class="container">
  <div class="row row--grid">
	<div class="col-xs-6 col-sm-4">
		...
	</div>
	<div class="col-xs-6 col-sm-4">
		...
	</div>
  </div>
</div>

then internal "col" at the resolution:
- less than 768px will have a distance of 10px
- more than 768px will have a distance of 20px


If you add to "row" modification "row--grid-big", for example:

<div class="container">
  <div class="row row--grid-big">
	<div class="col-xs-12 col-sm-4">
		...
	</div>
	<div class="col-xs-12 col-sm-4">
		...
	</div>
  </div>
</div>

then the internal "col" will have a distance of 40px

Home slider

To change the slider image, open HTML file and replace the value of data-bg attribute of this construction:

<div class="item home-carousel__item" data-bg="img/home/slide1.jpg">
  <div class="home-carousel__text">
	...
  </div>
</div>

Logotype

To change header logotype, find image img/logo.png and replace it with your logo.

To change footer logotype, find image img/logo--white.png and replace it with your logo.
